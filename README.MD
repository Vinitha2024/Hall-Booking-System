This is a simple console-based Hall Booking System written in Java using a single file (Main.java).

The program allows users to:
    Select a role as 1,2,3,4,5(Developer, Manager, Admin, etc.)
    Enter number of participants
    Enter a valid meeting date
    Choose a time slot (Morning / Evening)
    View available halls
    Book a hall without double booking

Note:
--->Past dates are not allowed
---->Halls above 20 capacity are restricted based on role
---->Morning and Evening are treated as separate slots


Procedure to run the program:
--->javac Main.java
--->java Main
